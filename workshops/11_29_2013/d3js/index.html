What is D3?

D3plus is a library built to take advantage of the robust featureset of D3, while providing a fairly low barrier of entry to users new to code, design, and/or data visualization.

How to use D3

Selecting a DOM element

d3.select() will select a DOM element based on standard selector strings. In this example, we are creating a D3js element from a DIV with an ID of "container". You can read more about D3 selections here.

Adding a DOM element to the page

.append() will add a new element to the page. In this example, we are creating a "div" node to add to the page. You can read more about how this works here.

Binding Data to DOM elements

So far everything we've done can be done via jQuery. .data() will enable the user to bind data to specific elements on the page thus driving their appearance or disappearance. In this example, we are binding a simple array of values to our previous "div" node that is already on the page. You can read more about how the data operator here.
